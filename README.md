# webapi-seed

## Technologies
* Server Side
    * Asp.Net Web Api
    * Entity Framework
    * Castle Windsor
    * Autommaper
    * Moq
    
##Architecture
* DDD classic
    * Domain Services.
    * Repository pattern with Unit of Work per request.
    * Inversion of control using conventions .
    * Autommaping for custom views decoupled from domain.
